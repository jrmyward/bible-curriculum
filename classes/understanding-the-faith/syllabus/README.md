# Official Syllabus

This directory contains the official syllabus from the *Understanding the Faith* textbook.

## Contents

- **Images**: Original syllabus pages from the book (`.jpg`, `.png`, etc.)
- **`syllabus.md`**: Markdown version generated by `/generate-official-syllabus`

## Workflow

1. Add syllabus image(s) to this directory
2. Run `/generate-official-syllabus` to convert images → `syllabus.md`
3. Run `/generate-map` to create the teaching map using `syllabus.md` + school calendar
