# Understanding the Faith

**Grade Level:** 11–12
**Series:** *Understanding the Times* (Summit Ministries / David Noebel & Jeff Myers)
**Status:** Active — Teaching map complete

## Course Description

A foundational Bible course that explores the core doctrines of the Christian faith through the lens of worldview thinking. Students examine what Christians believe and why — engaging with theology, philosophy, ethics, and apologetics through a cohort-style, discussion-first model. The course uses the *Understanding the Times* text as its primary framework, supplemented by contemporary resources and guided discussion.

## Teaching Model

This course follows the cohort teaching tools defined in `_shared/cohort-tools.md`:

1. **Discussion Briefs** — Written before each discussion session
2. **Pair & Defend** — Structured debate on assigned positions
3. **Case Studies** — Real-world application in small groups
4. **Capstone Projects** — End-of-unit synthesis presentations
