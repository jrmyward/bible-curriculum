# Foundations — Course-Wide Publisher Docs

Authoritative course-level documents from the Summit portal. These validate and detail the
deduced [syllabus](../../../syllabus/syllabus.md) and [teaching map](../../../teaching-map.md).

| File | What it is | Notes |
| --- | --- | --- |
| `course-overview.pdf` | Course description + **full lesson-by-lesson outline** (all 10 subsections per chapter, e.g. `6.1 Creation Stories … 6.10 Conclusion`) | The map for building compressed lesson plans — the exact publisher lessons to merge into each 8-day block |
| `scope-and-sequence.pdf` | Official Key Themes + Key Verses per chapter | 2 pp. Confirms 18 chapters × 10 lessons = 180-day design |
| `teacher-resources-all-chapters.pdf` | All chapters' teacher-resource handouts + tests (`TR n.nA …`) in one file (5.6 MB) | Source for per-chapter tests; extract to markdown per chapter as needed |
| `presentation-slides-all-chapters.pdf` | Publisher slide deck, all lessons (`PS n.nA …`), 564 pp. (49 MB) | See note below re: committing |

## Confirmed facts (from these docs)

- **18 chapters, 10 lessons each = 180 publisher lessons** (compressed to ~8 days/chapter here).
- Course goal (publisher): understand the Christian worldview; the Bible's authority; the
  Trinity; the Bible as **a metanarrative in six acts and an interlude**; loving God and others.
- **Ch 1 is a whole-course preview** — its subsections each foreshadow a later part
  (creation→Israel, Jesus→new creation, who is God, becoming like Jesus).
- **Ch 18 (Conclusion) has only 5 subsections** — confirms it's a shorter review chapter.

## Note on the slide deck

`presentation-slides-all-chapters.pdf` is 49 MB / 564 pp. It is **kept local, not committed**
— large slide decks (`*slides*.pdf`, `*.pptx`, `*.key`) under `portal/` are gitignored to keep
the repo lean. The file lives here on disk; re-download from the portal if it's ever lost.
