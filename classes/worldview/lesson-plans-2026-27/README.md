## Structure

Lesson plans are organized by calendar week (following the `/scaffold-lesson-structure` convention) — each week has:

- `README.md` — week overview (what unit, key activities, assessments due)
- `1-monday.md` through `5-friday.md` — daily lesson plans (number prefix preserves order)
- `substitute-plan-<day>.md` — when a sub is required (teacher-absence day)
- `handouts/` — week-specific handouts
